# Ingreso_app

_Breve aplicación de escritorio que consiste en agregar usuario, contraseña e imagen.

***

## Información general :pencil:

:heavy_check_mark: Pantalla principal, agregar y buscar usuarios almacenados en una base de datos con C# y SQLServer.<p>
:heavy_check_mark: La información debe quedar almacenado en una base de datos.<p>
:heavy_check_mark: Estado del proyecto: Finalizado.

:mag_right: NOTA: la parte del diseño puede mejorarse.

## Instalación

_Una pequeña introducción sobre la instalación:
  
**The full git repository is at: <https://github.com/edwsilme/ingreso_app>

```
$ git clone https://github.com/edwsilme/ingreso_app.git
$ cd ../path/to/the/file
$ npm install
$ npm start
```

... El script de la base de datos se encuentra almacenado en la carpeta script. :sparkles:
 
 ... Modificar en el código:<p>
 <img src="https://raw.githubusercontent.com/edwsilme/ingreso_app/main/img/class.JPG" width="300"><p>
